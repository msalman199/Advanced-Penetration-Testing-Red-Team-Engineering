# 🔧 Troubleshooting

## ⚠️ Maigret Not Found

### Solution

```bash
export PATH=$PATH:~/.local/bin
```

- 🔄 Verify pip installation
- 🔄 Restart terminal session

---

## ⚠️ Search Timeouts

### Solution

- ⏱️ Increase timeout values
- ⏱️ Reduce request volume
- ⏱️ Add delays between searches

---

## ⚠️ JSON Processing Errors

### Solution

- 📄 Validate JSON syntax
- 🛠️ Add exception handling
- 🔍 Verify file permissions

---

## ⚠️ Empty Results

### Solution

- 🌐 Verify internet connectivity
- 🔍 Enable verbose mode
- 🔄 Retry investigation
